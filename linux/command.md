## Linux(권한 개 빡셈)
CD - Change directory

CD / --> 맨 위로 상위 쪽으로 가는거 root<br>
CD ~ --> home으로 가는거임<br>ex)
/ 에서 만들면 권한이 없는 일반 사용자는 사용 X <br>
~ (home) 에서 만들면 권한이 없어도 다 이용 쌉가능 ㅇㅇ 약간 차이점 가지고 있음
